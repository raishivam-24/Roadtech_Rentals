# Hero video

Drop the client's hero video here as: hero-banner.mp4

Recommendations to pass to the client:
- Landscape, 1920x1080 or higher
- 15-30 seconds, loopable (no jarring start/end cut)
- Machine actively working (milling, paving, stabilizing) — not static/parked shots
- Keep file size reasonable (under 15MB) for fast page load; compress with HandBrake or similar if the original is large
- No audio needed — video plays muted in the hero

Until hero-banner.mp4 is added, the hero shows the poster image
(public/images/hero/hero-banner.jpg) as a static fallback.
